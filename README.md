# 批量文件重命名工具 v2.0

一个功能强大的 Windows 桌面批量文件重命名软件，支持多种重命名模式的自由组合。

## 功能特性

### 核心功能
- **倒序排列文件** - 反转文件列表顺序
- **删除指定文字** - 从文件名中删除特定文字
- **查找替换** - 支持普通替换和正则表达式替换
- **添加前缀/后缀** - 在文件名前后添加指定文字
- **序号重命名** - 按序号批量重命名（支持补零）
- **大小写转换** - 全部小写、大写、首字母大写、驼峰命名
- **修改扩展名** - 批量修改文件扩展名

### 特色功能
- **多选组合** - 可同时启用多个重命名功能，按顺序依次处理
- **实时预览** - 执行前预览重命名效果，绿色背景显示将被修改的文件
- **撤销功能** - 支持撤销上一次重命名操作
- **文件过滤** - 按扩展名或关键字过滤文件
- **支持文件和文件夹** - 可选择单个文件夹或多个文件进行重命名

## 使用方法

### 方式一：直接运行 EXE（推荐）
1. 下载 `批量文件重命名工具.exe`
2. 双击运行
3. 选择文件夹或文件
4. 勾选需要的功能并填写参数
5. 点击"预览效果"查看结果
6. 确认无误后点击"执行重命名"

### 方式二：运行 Python 脚本
1. 确保已安装 Python 3.6+
2. 安装依赖：`pip install pillow`（仅生成图标需要）
3. 运行：`python batch_rename.py`

## 编译 EXE

如需自行编译 EXE：

```bash
# 安装 PyInstaller
pip install pyinstaller pillow

# 生成图标（可选）
python create_icon.py

# 编译 EXE
pyinstaller --onefile --windowed --icon=app_icon.ico --name="批量文件重命名工具" batch_rename.py
```

编译完成后，EXE 文件位于 `dist` 目录中。

## 注意事项

1. **序号重命名**时，基础名称不能为空
2. 重命名操作会直接修改文件名，建议先使用"预览效果"确认
3. 支持撤销功能，但只能撤销上一次操作
4. 如果目标文件已存在，会跳过该文件

## 系统要求

- Windows 7/8/10/11
- 无需安装 Python（EXE 版本）
